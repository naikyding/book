## git 設定

### 查詢 git config

- 查詢目前 git 所有設定

  ```sh
  git config --list
  ```

### **Personal 個人資料**

- 設定作者姓名

  ```sh
  git config --global user.name "Naiky"
  ```

- 設定電子信箱

  ```sh
  git config --global user.email "74ding@gmail.com"
  ```
