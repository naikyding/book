## Setting

- **Personal 個人資料**
  - 查詢 git 設定內容 `git config --list`
  - 輸入作者姓名 `git config --global user.name "Naiky"`
  - 輸入電子信箱 `git config --global user.email "74ding@gmail.com"`

## local repo 本地資料庫
